# FRIDAY Heartbeat Config
# Edit this file to control what FRIDAY checks in the background.
# Plain English — FRIDAY reads this on every tick.

## Schedule
- Check every 30 minutes
- Quiet hours: 1am — 7am (no alerts)
- Max 3 proactive messages per day

## Checks
- Any critical/urgent emails unread for more than 2 hours?
- Any monitor alerts queued and undelivered?
- Is it 8am on a weekday? Run the morning briefing.

## Alert Style
- Keep it to 1-2 sentences max
- Only alert if something actually needs attention
- If nothing urgent, stay silent
