## Posting Notes

**X algorithm tips:**
- No external links in main tweet (kills reach)
- Video/image gets 2-3x engagement
- Reply thread immediately — keeps engagement on your post
- Post between 2-5pm GMT for UK/US overlap
- "I built" hook outperforms "check out my project"

**LinkedIn algorithm tips:**
- No links in the post body — put in first comment
- Story format > feature list (open with the problem)

- End with a question (drives comments, comments = reach)
- First 3 lines are the hook — they show before "see more"
- Post Tuesday-Thursday 8-10am GMT for max reach
